# R2 Parquet Downloader

Simple Windows desktop application for mirroring Parquet files
from Cloudflare R2 to a local folder.

## Features
- One-click refresh
- Download only new files
- No installation required
- Read-only access to R2 bucket

## Download
➡️ [Download latest EXE](https://github.com/xxx/xxx/releases/latest)

## First run
1. Enter R2 credentials
2. Choose local destination folder
3. Click Save & Test

## Security
- Credentials are stored locally in Windows AppData
- Read-only bucket access recommended
